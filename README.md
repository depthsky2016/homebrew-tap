# homebrew-tap

Homebrew tap for Jiang's apps.

## Install

```bash
brew tap depthsky2016/tap
brew install --cask voiceinput
```

Or in one line:

```bash
brew install --cask depthsky2016/tap/voiceinput
```

## Casks

| Cask | App |
|------|-----|
| `voiceinput` | [VoiceInput 声忆](https://voiceinput.app/) — macOS menu-bar voice input |
